# EnkaProfileCard

基于 [Enka.Network API](https://api.enka.network/) 的游戏个人信息卡片生成工具

支持游戏：`原神`、~~`崩坏：星穹铁道`~~（开发中）

Enka.Network 使用游戏内部 API 获取数据，与米游社（HoYoLab）API 不同，无法获取活跃天数、探索度、宝箱数量等数据
